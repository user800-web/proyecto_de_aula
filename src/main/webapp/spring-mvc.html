Spring MVC

Introducción

Spring MVC es un módulo en Spring Framework (o Spring para abreviar) para aplicaciones web de desarrollo rápido. El MVC en Spring M VC significa Model-View-Controller, un patrón de diseño ampliamente utilizado en el desarrollo de la interfaz gráfica de usuario (GUI). Este patrón no solo es común en el desarrollo web, sino que también se usa en tecnología de escritorio como Swing y JavaFX.

A veces llamado Spring Web MVC, Spring MVC es uno de los marcos web más populares en la actualidad y una de las habilidades más buscadas. Esta información es para los que quieran aprender a desarrollar aplicaciones web basadas en Java con Spring MVC.

El Protocolo de Transferencia de Hipertexto (HTTP)

El Protocolo de transferencia de hipertexto (HTTP) permite que los servidores web y los navegadores intercambien datos a través de Internet o una intranet. El World Wide Web Consortium (W3C), una comunidad internacional que desarrolla estándares, es responsable de revisar y mantener este protocolo. La primera versión de fue HTTP 0.9, que luego fue reemplazada por HTTP 1.0. El reemplazo de HTTP 1.0 es HTTP 1.1, que se define en la Solicitud de comentarios (RFC) 2616 del W3C. Al momento de escribir este artículo, HTTP 1.1 sigue siendo la versión más popular de HTTP. La versión actual, HITP/2, se lanzó en mayo de 2015.
